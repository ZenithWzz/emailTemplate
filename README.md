# Email Template Builder

Nuxt 3 + TypeScript + Vuetify web app สำหรับสร้างและปรับแต่งอีเมลเทมเพลตได้อย่างอิสระ พร้อมพรีวิวแบบสดและปุ่มคัดลอก/ดาวน์โหลด HTML

## คุณสมบัติหลัก
- เพิ่มบล็อกข้อความ รูปภาพ ปุ่ม หรือเส้นคั่น และจัดเรียงลำดับได้ง่าย
- ปรับความกว้างอีเมล สีพื้นหลัง ฟอนต์ ระยะขอบ และ CSS เฉพาะเพิ่มเติม
- ปรับสไตล์ของแต่ละบล็อกได้ เช่น จัดวางข้อความ สี ขนาดตัวอักษร มุมโค้ง และ padding
- คัดลอกหรือดาวน์โหลดไฟล์ HTML ที่สร้างขึ้นพร้อมสไตล์ inline

## การเริ่มต้นใช้งาน
1. ติดตั้ง dependencies
   ```bash
   npm install
   ```
2. รันโหมดพัฒนา
   ```bash
   npm run dev
   ```
3. เปิดเบราว์เซอร์ที่ `http://localhost:3000` เพื่อเริ่มใช้งาน

## คำสั่งเพิ่มเติม
- สร้างไฟล์ production: `npm run build`
- พรีวิวหลัง build: `npm run preview`

> หมายเหตุ: โปรเจกต์ตั้งค่า `ssr: false` เพื่อให้ทำงานเป็นแอปแบบ SPA สำหรับเครื่องมือแก้ไขเทมเพลต
